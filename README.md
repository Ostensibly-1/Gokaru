# Gokaru ♟
## A simple Chess-client built with Go and Fyne UI Toolkit
### ⚠ Made in about 2 days, if there are any bugs found, file an issue or a pull request if you have a solution
Notes: this was a tiny mini project that I made to relearn Go and learn the Fyne UI toolkit which I will use in future projects
## How to build
1. Download go from https://go.dev/
2. Setup Fyne from https://fyne.io/
3. Run `go mod tidy` on the project directory
4. Run `go build .` and wait for the executable
5. Run `gokaru.exe` or whatever the build name was and enjoy
